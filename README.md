# Projekt Kryptograficzny
Program zawiera:
- Szyfr Cezara
   - Szyfr Cezara to prosty szyfr w , którym każdą literę tekstu zastępuje się literą przesuniętą o stałą liczbe pozycji w alfabecie.
- Szyfr Vigenère
     - Szyfr Vigenère’a to klasyczna, wieloalfabetowa metoda szyfrowania podstawieniowego, która wykorzystuje słowo-klucz do wielokrotnego stosowania szyfru Cezara.
## Funkcjonalności
1. Szyfrowanie i deszyfrowanie tekstu przy użyciu szyfru Cezara.
2.  Szyfrowanie i deszyfrowanie tekstu przy użyciu szyfru Vigenère.
3.  Generowanie losowych kluczy
4.  Próba łamanie zaszyfrowanych wiadomości
